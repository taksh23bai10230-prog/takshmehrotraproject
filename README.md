# Smart Vision System

## 📌 Overview
This project is a real-time computer vision system built using OpenCV and Python. It performs object detection, tracking, image processing, and segmentation.

---

## 🚀 Features
- Image preprocessing
- Edge and corner detection
- Morphological operations
- Image segmentation
- Object detection (YOLO)
- Object tracking

---

## 🛠️ Tech Stack
- Python
- OpenCV
- NumPy

---

## ⚙️ Installation

1. Clone the repository:
git clone https://github.com/your-username/Smart-Vision-System.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the project:
python src/main.py
